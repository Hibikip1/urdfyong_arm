# urdfyong_moveit_config — MoveIt2 Configuration

Auto-generated by MoveIt Setup Assistant + hand-tuned for ROS2 Humble.

## Quick Start

```bash
# MoveIt2 demo (RViz + interactive marker + motion planning)
ros2 launch urdfyong_moveit_config demo.launch.py
```

Drag the orange interactive marker in RViz2, click "Plan" to see trajectories.

## Key Config Files

| File | Purpose |
|------|---------|
| `urdfyong.srdf` | Planning group (`arm_group`), home pose, collision matrix |
| `kinematics.yaml` | KDL solver: 0.005s resolution, 0.05s timeout, 3 attempts |
| `moveit_controllers.yaml` | MoveIt→Controller bridge (`FollowJointTrajectory` action) |
| `ros2_controllers.yaml` | Controller manager: JTC + JSB config |

## Critical: `moveit_controllers.yaml`

The `moveit_controller_manager` plugin declaration **must** be present. Without it, move_group segfaults with:
```
Parameter '~moveit_controller_manager' not specified
```

## Launch Files

| Launch | Description |
|--------|-------------|
